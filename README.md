* ido
personal nodeproject based on express
          使用dwz-jui框架搭建的一个快速开发平台
=========
(https://github.com/qimuyunduan/blogme)

###代码相关
* 前台：jquery,dwz
* 后台：node.js 
* 适配于dwz框架的GridTree版本


###功能模块
* 登录验证
* 用户管理
* 组织架构
* 菜单管理
* 缓存管理
* 文件上传
* 权限控制(1、用户-菜单权限;2、角色权限管理;3、人员角色授权)


###后续功能
* 首页门户
* 消息通知弹窗
